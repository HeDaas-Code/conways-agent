# Obsidian Agent

基于 Obsidian vault 的拟人化自发展认知体。

## 架构

- `/python/` — Agent 核心引擎（Brain）：LLM 推理、世界观处理、目标系统、演化系统
- `/node/` — Agent 感知与交互层（Body）：文件监听、Obsidian 插件、终端 UI
- `/agent/` — Agent 的 vault 空间：人格、记忆、目标、世界观、知识库
