## Running the Agent

```bash
# Set environment variable
export OBSIDIAN_VAULT_PATH=/path/to/your/vault

# Or create a .env file
echo "OBSIDIAN_VAULT_PATH=/path/to/your/vault" > .env

# Run the agent
cd python
python -m agent.main
```
